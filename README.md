Stylelint config for Ant international FE team

## Installment

```bash
$ npm i stylelint-config-antife --save-dev
```

## Usage

Add `.stylelintrc.js` in your root directory.

```javascript
module.exports = {
  extends: 'stylelint-config-antife',
}
```
